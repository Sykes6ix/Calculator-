# 🧮 Responsive Calculator App

A simple and responsive calculator built using **HTML, CSS, and JavaScript**.
This project performs basic arithmetic operations and works smoothly across different screen sizes.

---

## 🚀 Live Demo

👉 https://sykes6ix.github.io/Calculator-/



---

## 🚀 Features

* ➕ Addition, ➖ Subtraction, ✖️ Multiplication, ➗ Division
* 📱 Fully responsive design (mobile, tablet, desktop)
* ⚡ Fast and lightweight
* 🎨 Clean and user-friendly interface
* ⌨️ Button-based input system

---

## 🛠️ Technologies Used

* **HTML5** – Structure
* **CSS3** – Styling & responsiveness
* **JavaScript (Vanilla JS)** – Logic and functionality
